# Stage 2 — native RCF-in-ZFC: scope (the big lever, now bounded)

## Goal

Collapse the **transport bridge** (~10^46, EXTRACTED) — the only large
number left — by formalizing a model of F1 *natively in ZFC* and binding
F1-satisfaction to **set.mm's genuine ZF base** (the shared 13-axiom

analytic ℝ. set.mm's ~10^46 is an artifact of *its* construction
choices; a native ZF model pays only the ZF-set construction, not the
analytic-ℝ tower.

## The scope collapse (why this is now bounded, not open-ended)

Stage 1 MEASURED that the closed ASA′ proof forces exactly **one
distinct, un-nested radical**: `( sqrt ( u * ( inv ( sqd a c ) ) ) )`,
K = 1, nesting depth 1. Consequence: the model the proof needs is **not**
the full Euclidean closure of ℚ and **not** real-closed-field theory.
It is:

    ℚ_geo  ⊕  exactly ONE quadratic extension  ℚ_geo(√r)

where `r` is that single measured radicand. This removes the open-ended
"formalize all of RCF" risk. We formalize a *finite, specific* tower of
height 1 over a directly-built ℚ.
